Приложение считывает исходные данные из excel-файлов, анализирует их и генерирует на основе этих данных расписание работы оборудования выводя его на экран.
Также приложение ищет оптимальные варианты с точки зрения временных затрат.
Excel-файлы располагаются в папке "xlsx_data", которая находится в папке с проектом.
Доступен экспорт расписания в xlsx-формат. Экспортированный файл помещается в папку "output".
